### Visual Ratio

A lightweight, client-side Minecraft mod that allows you to change the game's **aspect ratio** independently of the window or screen resolution.

### Features 

- Change aspect ratio with keybinds 
- Configurable step size
- No impact on gameplay mechanics

### Building from Source

```sh
./gradlew build
```

Jar appears in build/libs/

### Config 

Config file is locate in `config/visual-ratio.toml`.

Example:

```
enable = false
aspectRatio = 1.5
ratioStep = 0.5
```
